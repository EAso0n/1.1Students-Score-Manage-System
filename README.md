### 第 1 周：Python 语法速通

**核心认知：你有 Java 基础，Python 语法一周足够。难点不是语法本身，是动态类型的思维切换。**

| 天数  | 内容                     | 廖雪峰对应章节                           | 你的 Java 对应                                        |
| ------- | -------------------------- | ------------------------------------------ | ------------------------------------------------------- |
| Day 1 | 环境安装 + 变量/数据类型 | Python 解释器、数据类型和变量            | `int a = 1;` → `a = 1`（去掉了类型声明）                                |
| Day 2 | 字符串/列表/字典         | 字符串和编码、list 和 tuple、dict 和 set | `String` → `str`，`ArrayList` → `list`，`HashMap` → `dict`                                      |
| Day 3 | 条件判断/循环            | 条件判断、循环                           | `for(int i=0;i<n;i++)` → `for i in range(n):`                                                  |
| Day 4 | 函数/参数传递            | 函数、函数的参数                         | 重点理解：Python 的参数传递是"引用传递"，和 Java 不同 |
| Day 5 | 面向对象/类              | 面向对象编程                             | `class` 语法对比 Java 的差异，重点关注 `self` 和 `__init__`                  |
| Day 6 | 异常处理/文件操作        | 错误处理、IO 编程                        | `try-catch` → `try-except`                                                  |
| Day 7 | 综合练习                 | —                                       | 把之前所有内容串一遍                                  |

**学习方法：**

- 不要只看不写。每看完一节，立刻在 IDE 里写代码验证。廖雪峰的教程每章末尾有练习题，全部做完
- 禁止行为：不要边学 Python 边和 Java 对比语法差异——这会拖慢你。接受"Python 就是这样写的"
- Java→Python 核心思维切换（三句话记住）：① Python 不需要声明类型 ② Python 用缩进代替大括号 ③ Python 万物皆对象，函数也是对象

**检验标准：**   能用 Python 独立写出：一个学生成绩管理系统（增删改查 + 排序 + 文件保存），不需要查任何资料。做不出来就再来一周。
